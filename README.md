# ``BbAppEditPost``

Post editing functionality and user interface.

## Overview

The BbAppEditPost library provides comprehensive post editing capabilities. It includes the edit post view interface, intent sheet presentation for modal editing workflows, and the service layer that handles post modification operations. The module supports editing post titles, content, section assignments, and post settings while maintaining synchronization with content sources.
